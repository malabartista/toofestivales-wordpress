## REST API

Access the WordPress site's data through an easy-to-use HTTP REST API.

### REST API Developer Endpoint Reference

| ----------------- |:--------------------------------------:|
| Events	          | /wp-json/eventsmanager/v1/events/      |
| Events Categories | /wp-json/eventsmanager/v1/categories/  |
